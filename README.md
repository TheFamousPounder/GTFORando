# GTFORando
A web based set of randomizers for games I often play. 
Currently has randomizers for GTFO by 10 Chambers and The Outlast Trials by Red Barrels.

Current GTFO features: 

Loadout randomizer that randomizes Main Weapon, Special Weapon, Tool, and Melee. Settings to exclude weapons from each category in the randomizer, these settings are saved so that you do not need to reset them each time you use the site. A force reroll option which makes it so that each randomization is completely new featuring no repeats from the last randomization.

Rundown randomizer that randomly chooses an expedition to play. Settings for it include choosing which specific rundown is chosen from and/or checks for what type of objectives you want.

A cosmetic randomizer that randomizes every cosmetic available in the game, as well as a settings page for cosmetics too.

Multi-rando which is the same as normal loadout randomizer except it is ran 4 times, for use with multiple players or bots.

Current Outlast features:

Basic loadout randomizer for amps and rigs, will add more options soon.
